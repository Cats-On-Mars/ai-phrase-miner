# 短语工具 Chrome 插件安装说明

## 功能介绍
这是一个英语学习工具，可以帮助你：
- ✨ 生成单词相关的常用短语
- 📝 从文章中提取四级以上英语短语
- 📖 管理个人单词本
- 📄 保存短文到短文簿

## 安装步骤

### 方法一：开发者模式安装（推荐）

1. **解压文件**
   - 将收到的压缩包解压到任意文件夹
   - 记住这个文件夹的位置（不要删除）

2. **打开Chrome扩展程序页面**
   - 在Chrome浏览器地址栏输入：`chrome://extensions/`
   - 或者点击右上角三个点 → 更多工具 → 扩展程序

3. **开启开发者模式**
   - 在页面右上角找到"开发者模式"开关
   - 点击开启

4. **加载插件**
   - 点击左上角"加载已解压的扩展程序"按钮
   - 选择刚才解压的文件夹
   - 点击"选择文件夹"

5. **完成安装**
   - 插件会出现在扩展程序列表中
   - 可以固定到工具栏方便使用

### 方法二：Edge浏览器安装

1. 打开Edge浏览器，访问：`edge://extensions/`
2. 开启"开发人员模式"
3. 点击"加载解压缩的扩展"
4. 选择解压后的文件夹

## 使用说明

### 打开插件
- 点击浏览器右上角的插件图标
- 或者使用快捷键（如果设置了）

### 配置API Key
1. 首次使用时，点击插件右上角的齿轮按钮
2. 输入你自己的 DashScope API Key
3. 点击"保存"
4. API Key 只保存在本机浏览器存储中，不会写入项目文件

### 生成短语
1. 点击"生成"标签
2. 输入一个英文单词（如：advantage）
3. 点击"生成短语"按钮
4. 查看包含该单词的常用短语和例句

### 提取短语
1. 点击"提取"标签
2. 粘贴英文文章内容
3. 点击"提取短语"按钮
4. 自动提取文章中的四级以上短语

### 单词本
- 在生成或提取的短语旁边点击"+"按钮
- 短语会自动添加到单词本
- 可以按日期筛选查看
- 支持导出为Markdown文件

### 短文簿
- 提取短语后，文章会自动保存到短文簿
- 点击文章可以查看详情和提取的短语
- 可以删除不需要的文章

## 注意事项

1. **不要删除插件文件夹**
   - 安装后不要删除或移动解压的文件夹
   - 否则插件会失效

2. **API密钥**
   - 插件不内置API密钥
   - 需要在插件右上角齿轮按钮中填写自己的 DashScope API Key
   - 如果遇到调用失败，请检查网络、密钥是否有效，以及账户额度是否充足

3. **数据存储**
   - 所有数据（单词本、短文簿）都存储在本地
   - 卸载插件前记得导出重要数据

4. **浏览器兼容性**
   - 支持Chrome、Edge、Brave等基于Chromium的浏览器
   - 不支持Firefox、Safari

## 常见问题

**Q: 插件图标不显示？**
A: 点击浏览器右上角的拼图图标，找到"短语工具"并固定到工具栏

**Q: 提取/生成失败？**
A: 检查网络连接，或稍后重试。可能是API调用次数限制

**Q: 如何更新插件？**
A: 收到新版本后，重新解压并在扩展程序页面点击"重新加载"

**Q: 如何卸载？**
A: 在 `chrome://extensions/` 页面找到插件，点击"移除"
