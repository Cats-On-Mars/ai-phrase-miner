# 英语短语学习工具

一款基于 Chrome 侧边栏（Side Panel）的英语短语学习 Chrome 扩展，集成 AI 智能短语提取、沉浸式阅读、生词管理和间隔复习功能，帮助用户在日常浏览中高效学习英语短语和搭配。

---

## 核心功能

### 1. 生成模式

输入一个英文单词，AI 自动生成该单词的 3-8 个常用短语（四级以上水平）。

- 输入单词后点击"生成短语"
- 每个短语展示：短语本身、音标、中文释义、2 个例句
- 可将感兴趣的短语一键添加到单词本

### 2. 翻译模式（沉浸式阅读）

粘贴英文短文或上传文件（`.txt` / `.md`），AI 自动提取文中有学习价值的短语、搭配和高级词汇。

**短语提取：**
- AI 识别文中的动词搭配、介词搭配、固定表达、习语等
- 同时提取有学习价值的单个实义词（排除常见简单词）
- 根据上下文语境给出准确释义、词性、音标

**沉浸式阅读界面：**
- 原文中被识别的短语高亮显示（支持多词短语匹配）
- 鼠标悬停短语即显示浮动释义卡片（音标 + 词性 + 中文释义）
- 双击短语添加到生词区

**生词区：**
- 双击文中短语即可收录到生词区
- 点击"生成例句"，AI 为每个生词短语生成 2 个实用例句
- 生成例句后自动添加到单词本，无需手动操作
- 支持增量生成：新添加的生词再次点击按钮，只为新增部分生成例句并添加

### 3. 翻译记录

每次翻译自动保存为一条记录，方便日后回顾。

- 顶部右上角的时钟图标进入翻译记录页
- 记录列表按时间倒序排列，显示文章标题和日期
- 点击记录恢复到沉浸式阅读界面，包括原文、短语释义、生词区、已生成的例句
- 支持删除不需要的记录
- 从记录进入详情页后，返回按钮会回到记录列表（而非首页）

### 4. 单词本

统一管理所有收录的短语和词汇。

- 展示：短语、音标、释义、例句
- 按日期筛选：支持按添加日期查看
- 复习进度可视化：每个短语下方有 6 个圆点，表示学习和复习状态
- 支持导出和删除

### 5. 间隔复习系统

基于艾宾浩斯遗忘曲线的间隔复习机制，帮助长期记忆。

- **复习时间节点**：添加当天 → 第 1 天 → 第 3 天 → 第 6 天 → 第 14 天 → 第 29 天
- 自动顺延：如果错过了某天的复习，系统自动将后续复习计划顺延
- "今日复习"按钮显示待复习数量，点击进入复习模式
- 复习界面以卡片形式展示短语，勾选即标记为已复习
- 复习进度圆点：红色实心 = 已完成，灰色空心 = 待完成，悬停可查看具体日期和状态

## 技术架构

### 技术栈

| 组件 | 技术 |
|------|------|
| 扩展框架 | Chrome Extension Manifest V3 |
| UI 渲染 | 原生 JavaScript + 字符串模板 |
| 样式 | 纯 CSS（自定义设计，无框架） |
| AI 接口 | 阿里云通义千问 API（qwen3.7-plus） |
| 数据持久化 | Chrome Storage API（`chrome.storage.local`） |

### 文件结构

```
短语提取工具/
├── manifest.json          # 扩展配置（MV3）
├── background.js          # Service Worker，处理图标点击打开侧边栏
├── sidepanel.html         # 侧边栏页面入口
├── sidepanel.js           # 核心逻辑：状态管理、AI 调用、UI 渲染、事件绑定
├── styles.css             # 全部样式
├── icon16/48/128.png      # 扩展图标
├── home.png               # 首页导航图标
├── 单词库.png              # 单词本导航图标
└── 安装说明.md             # 安装指南
```

### 架构设计

采用**单状态树 + 全量渲染**的简洁架构：

- **状态管理**：全局 `state` 对象存储所有应用状态（模式、输入、短语、单词本、翻译记录等）
- **渲染机制**：每次状态变更后调用 `render()` 函数，重新生成整个 DOM 并绑定事件
- **数据持久化**：单词本、翻译记录通过 `chrome.storage.local` 持久存储，支持跨会话
- **API Key 配置**：项目不内置密钥，用户在插件设置中填写自己的 DashScope API Key，并保存到本机 `chrome.storage.local`
- **AI 调用**：所有 AI 功能通过阿里云通义千问 API（兼容 OpenAI 格式），支持分批/分段处理长文本

### 核心模块

| 模块 | 说明 |
|------|------|
| `batchTranslateWords()` | 分段发送文本给 AI，提取短语和释义 |
| `buildImmersiveHTML()` | 最长匹配算法，在原文中标记短语并生成可交互 HTML |
| `generateNewWordPhrases()` | 为生词区短语批量生成例句，自动添加到单词本 |
| `generatePhrases()` | 生成模式：为单个单词生成短语列表 |
| `addToVocabulary()` | 添加短语到单词本，初始化复习状态 |
| `markAsReviewed()` | 标记短语为已复习，更新复习计划 |
| `saveNewTranslateRecord()` | 保存翻译记录到 Chrome 存储 |
| `restoreTranslateRecord()` | 从记录恢复沉浸式阅读状态 |
| `render()` | 根据当前状态生成完整 UI 并绑定所有事件 |

## 使用流程

### 典型学习流程

```
粘贴英文文章 → AI 提取短语 → 沉浸式阅读
    ↓                              ↓
 自动保存记录              双击短语添加到生词区
                                   ↓
                           点击"生成例句"
                                   ↓
                        自动添加到单词本 + 间隔复习
```

### 导航说明

- **🏠 首页图标**：回到主页（生成/翻译模式切换）
- **📖 单词本图标**：查看所有收录的短语
- **🕐 时钟图标**：翻译记录列表

## 安装方式

1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 开启右上角"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本项目文件夹
5. 点击浏览器工具栏中的扩展图标，即可打开侧边栏
