# 英语短语学习工具

一个 Chrome / Edge 侧边栏插件，用于生成英语短语、从短文中提取表达、管理个人单词本，并按间隔复习节奏回顾短语。

## 功能特点

- ✅ 侧边栏显示，不影响浏览体验
- ✅ 输入单词生成常用短语、释义、音标和例句
- ✅ 支持自动提取：粘贴文章后直接生成短语列表、释义和例句
- ✅ 支持手动提取：粘贴英文短文进行沉浸式阅读
- ✅ 双击高亮词条加入生词区，并批量生成例句
- ✅ 单词本本地保存，支持日期筛选、导出 Markdown 和删除
- ✅ 提取记录本地保存，支持恢复自动提取结果和阅读现场
- ✅ 间隔复习进度展示和今日复习
- ✅ 支持 OpenAI 兼容的 `chat/completions` 接口，可使用百炼、DeepSeek、OpenRouter、中转服务或本地兼容服务

## 安装

### Chrome 开发者模式安装

1. 将项目文件夹放在固定位置，不要在安装后删除或移动
2. 打开 Chrome，访问 `chrome://extensions/`
3. 打开右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹
6. 插件安装完成后，可以在浏览器工具栏固定插件图标

### Edge 安装

1. 打开 Edge，访问 `edge://extensions/`
2. 开启"开发人员模式"
3. 点击"加载解压缩的扩展"
4. 选择本项目文件夹

## 使用说明

### 打开插件

1. 点击浏览器工具栏中的插件图标
2. 侧边栏会自动打开

### 配置 AI 接口

1. 首次使用时，点击插件右上角齿轮按钮
2. 填写 API 地址、模型名称和自己的 API Key
3. 点击"保存"
4. 默认 API 地址是阿里云百炼 DashScope，也可以换成其他 OpenAI 兼容接口
5. API 配置只保存在本机浏览器存储中，不会写入项目文件

常见配置示例：

| 平台 | API 地址 | 模型名称 |
|------|----------|----------|
| 阿里云百炼 | `https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions` | `qwen3.7-plus` |
| DeepSeek | 填平台提供的 `chat/completions` 地址 | 填对应模型名 |
| OpenRouter / 中转服务 | 填对应的 OpenAI 兼容地址 | 填对应模型名 |
| 本地兼容服务 | `http://localhost:端口/.../chat/completions` | 填本地服务模型名 |

### 生成短语

1. 点击"生成"标签
2. 输入一个英文单词，如 `advantage`
3. 点击"生成短语"
4. 查看包含该单词的常用短语和例句

### 自动提取

1. 点击"提取"标签
2. 粘贴英文文章内容
3. 点击"自动提取"
4. 查看自动生成的短语列表、释义和例句

### 手动提取

1. 点击"提取"标签
2. 粘贴英文短文内容
3. 点击"手动提取"
4. 阅读高亮后的文章，鼠标悬停查看释义
5. 双击短语或单词，将它加入生词区

### 单词本

- 在生成结果旁边点击"+"按钮，或在沉浸式阅读中生成生词例句
- 自动提取结果不会全量自动加入单词本，按需点击"+"添加
- 沉浸式阅读中为生词生成例句后，会自动添加到单词本
- 支持按日期筛选、导出 Markdown 和删除

### 提取记录

- 每次自动提取或手动提取都会自动保存到提取记录
- 点击右上角时钟按钮进入记录列表
- 点击记录可以恢复自动提取结果，或恢复手动提取的阅读现场
- 可以删除不需要的记录

### 今日复习

- 进入单词本后点击"今日复习"
- 勾选短语即可标记为已复习
- 复习计划会按添加当天、第 1 天、第 3 天、第 6 天、第 14 天、第 29 天推进

## 调试方法

如果侧边栏显示空白：

1. 右键点击侧边栏 → 选择"检查"打开开发者工具
2. 查看Console标签页中的错误信息
3. 检查Network标签页确认API请求是否成功
4. 确保API密钥有效且有足够的额度

## 注意事项

1. 项目不内置 API Key，用户需要在插件设置中填写自己的 AI 接口配置
2. API Key 和接口配置只保存在本机 `chrome.storage.local` 中，请勿提交到代码仓库
3. API 调用可能产生费用，请注意使用量
4. 为了支持自定义 HTTPS 接口和本地接口，插件会请求对应网络访问权限
5. 支持 Chrome、Edge、Brave 等基于 Chromium 的浏览器，不支持 Firefox、Safari

## 技术栈

- 原生JavaScript（无需构建工具）
- 纯CSS
- Chrome Extension Manifest V3
- OpenAI 兼容 `chat/completions` 接口（默认百炼 `qwen3.7-plus`）

## 常见问题

**Q: 插件图标不显示？**

A: 点击浏览器右上角的拼图图标，找到"短语工具"并固定到工具栏。

**Q: 生成/提取失败？**

A: 检查网络连接、API 地址、模型名称、API Key 和账户额度。

**Q: 如何更新插件？**

A: 拉取或替换新版本文件后，在扩展程序页面点击"重新加载"。

**Q: 如何卸载？**

A: 在 `chrome://extensions/` 页面找到插件，点击"移除"。
