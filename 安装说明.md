# 短语工具 Chrome 插件安装说明

## 功能介绍
这是一个英语学习工具，可以帮助你：
- ✨ 生成单词相关的常用短语
- 📝 粘贴英文文章或上传文件，进行沉浸式阅读
- 📖 管理个人单词本
- 🕐 保存和恢复翻译记录
- 🔁 按间隔复习节奏回顾短语

## 安装步骤

### 方法一：开发者模式安装（推荐）

1. **解压文件**
   - 将收到的压缩包解压到任意文件夹
   - 记住这个文件夹的位置（不要删除）

2. **打开Chrome扩展程序页面**
   - 在Chrome浏览器地址栏输入：`chrome://extensions/`
   - 或者点击右上角三个点 → 更多工具 → 扩展程序

3. **开启开发者模式**
   - 在页面右上角找到"开发者模式"开关
   - 点击开启

4. **加载插件**
   - 点击左上角"加载已解压的扩展程序"按钮
   - 选择刚才解压的文件夹
   - 点击"选择文件夹"

5. **完成安装**
   - 插件会出现在扩展程序列表中
   - 可以固定到工具栏方便使用

### 方法二：Edge浏览器安装

1. 打开Edge浏览器，访问：`edge://extensions/`
2. 开启"开发人员模式"
3. 点击"加载解压缩的扩展"
4. 选择解压后的文件夹

## 使用说明

### 打开插件
- 点击浏览器右上角的插件图标
- 或者使用快捷键（如果设置了）

### 配置API Key
1. 首次使用时，点击插件右上角的齿轮按钮
2. 输入你自己的 DashScope API Key
3. 点击"保存"
4. API Key 只保存在本机浏览器存储中，不会写入项目文件

### 生成短语
1. 点击"生成"标签
2. 输入一个英文单词（如：advantage）
3. 点击"生成短语"按钮
4. 查看包含该单词的常用短语和例句

### 沉浸式阅读
1. 点击"翻译"标签
2. 粘贴英文文章内容，或上传 `.txt` / `.md` 文件
3. 点击"开始沉浸式阅读"按钮
4. 阅读高亮后的文章，鼠标悬停查看释义
5. 双击短语或单词，将它加入生词区

### 单词本
- 在生成结果旁边点击"+"按钮，或在沉浸式阅读中生成生词例句
- 短语会自动添加到单词本
- 可以按日期筛选查看
- 支持导出为Markdown文件

### 翻译记录
- 每次沉浸式阅读会自动保存到翻译记录
- 点击右上角时钟按钮进入记录列表
- 点击记录可以恢复原文、高亮词条、生词区和已生成例句
- 可以删除不需要的记录

### 今日复习
- 进入单词本后点击"今日复习"
- 勾选短语即可标记为已复习
- 复习计划会按添加当天、第 1 天、第 3 天、第 6 天、第 14 天、第 29 天推进

## 注意事项

1. **不要删除插件文件夹**
   - 安装后不要删除或移动解压的文件夹
   - 否则插件会失效

2. **API密钥**
   - 插件不内置API密钥
   - 需要在插件右上角齿轮按钮中填写自己的 DashScope API Key
   - 如果遇到调用失败，请检查网络、密钥是否有效，以及账户额度是否充足

3. **数据存储**
   - 所有数据（单词本、翻译记录、API Key）都存储在本地浏览器
   - 卸载插件前记得导出重要数据

4. **浏览器兼容性**
   - 支持Chrome、Edge、Brave等基于Chromium的浏览器
   - 不支持Firefox、Safari

## 常见问题

**Q: 插件图标不显示？**
A: 点击浏览器右上角的拼图图标，找到"短语工具"并固定到工具栏

**Q: 生成/翻译失败？**
A: 检查网络连接，或稍后重试。可能是API调用次数限制

**Q: 如何更新插件？**
A: 收到新版本后，重新解压并在扩展程序页面点击"重新加载"

**Q: 如何卸载？**
A: 在 `chrome://extensions/` 页面找到插件，点击"移除"
