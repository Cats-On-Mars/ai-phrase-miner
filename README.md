# 英语短语学习工具 - Chrome浏览器插件

这是一个 Chrome 侧边栏插件，用于生成英语短语、沉浸式阅读英文文章、管理个人单词本，并按间隔复习节奏回顾短语。

## 功能特点

- ✅ 侧边栏显示，不影响浏览体验
- ✅ 输入单词生成常用短语、释义、音标和例句
- ✅ 粘贴英文文章或上传 `.txt` / `.md` 文件进行沉浸式阅读
- ✅ 自动识别短语、搭配和有学习价值的词汇
- ✅ 双击高亮词条加入生词区，并批量生成例句
- ✅ 单词本本地保存，支持日期筛选、导出 Markdown 和删除
- ✅ 翻译记录本地保存，支持恢复阅读现场
- ✅ 间隔复习进度展示和今日复习

## 安装步骤

### 1. 加载插件到Chrome

1. 打开Chrome浏览器
2. 在地址栏输入 `chrome://extensions/`
3. 打开右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择包含这些文件的文件夹
6. 插件安装完成！

### 2. 使用插件

1. 点击Chrome工具栏中的插件图标
2. 侧边栏会自动打开
3. 点击右上角齿轮按钮，填写自己的 DashScope API Key 并保存
4. 在"生成"标签输入单词，或在"翻译"标签粘贴英文文章
5. 查看短语结果、沉浸式阅读、生词区、单词本和复习记录

## 调试方法

如果侧边栏显示空白：

1. 右键点击侧边栏 → 选择"检查"打开开发者工具
2. 查看Console标签页中的错误信息
3. 检查Network标签页确认API请求是否成功
4. 确保API密钥有效且有足够的额度

## 文件说明

- `manifest.json` - Chrome插件配置文件
- `background.js` - 后台服务脚本
- `sidepanel.html` - 侧边栏HTML页面
- `sidepanel.js` - 纯JavaScript实现（无需React）
- `styles.css` - 插件样式
- `README.md` - 说明文档
- `安装说明.md` - 面向用户的安装和使用说明
- `项目介绍.md` - 功能和架构介绍

## 注意事项

1. 项目不内置 API Key，用户需要在插件设置中填写自己的 DashScope API Key
2. API Key 只保存在本机 `chrome.storage.local` 中，请勿提交到代码仓库
3. 确保网络连接正常
4. API调用可能产生费用，请注意使用量
5. 插件需要访问 `https://dashscope.aliyuncs.com/*` 的权限

## 技术栈

- 原生JavaScript（无需构建工具）
- 纯CSS
- Chrome Extension Manifest V3
- 阿里云通义千问API (qwen3.7-plus)
